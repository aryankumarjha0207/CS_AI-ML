#  Smart Expense Categorizer

A simple AIML-based project that classifies daily expenses into categories using Python and Machine Learning.

---

##  Features
- Classifies expenses (Food, Travel, Bills, Entertainment)
- Uses TF-IDF + Naive Bayes
- CLI-based interface
- Generates expense summary

---

##  Tech Stack
- Python
- Scikit-learn
- Pandas
